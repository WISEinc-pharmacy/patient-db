# 患者データベース

ワイズ株式会社の調剤薬局グループ向け患者情報管理ツール。  
HTML単一ファイル + Firebase で構成されたWebアプリケーション。

## 公開URL

https://wiseinc-pharmacy.github.io/patient-db/

## 技術スタック

| 項目 | 内容 |
|------|------|
| フロントエンド | HTML単一ファイル（CSS・JS内包） |
| データベース | Firebase Firestore（wise-patientdb） |
| 認証 | Google OAuth（Firebase Authentication） |
| ホスティング | GitHub Pages（legacyモード） |

## 機能一覧

### 認証・権限管理
- Googleアカウントによるログイン
- 3段階の権限管理（管理者 / 編集者 / 閲覧者）
  - **管理者**: 全機能 + 権限管理
  - **編集者**: 閲覧 + 登録・編集・削除 + CSVインポート
  - **閲覧者**: 閲覧 + CSV出力のみ

### 患者情報管理
- 新規登録・編集・削除（CRUD）
- 管理項目: 患者ID, 氏名（漢字）, 氏名（カナ）, 生年月日, 性別, 郵便番号, 住所, 電話番号, 担当店舗, 備考

### 検索・フィルター
- フリーワード検索（氏名・カナ・住所・電話番号）
- 店舗フィルター（柏, 足立, ながれやま, アイ調剤, 我孫子, さくら台）

### データ入出力
- **CSVインポート**: ヘッダー自動マッピング、患者IDで既存上書き/新規追加
- **CSV出力**: フィルター適用済みデータをエクスポート

### UI
- テーブルヘッダー固定（スクロール時も列名表示）
- レスポンシブ対応
